## Project Overview

This project is a **visually interactive weather monitoring dashboard** built using **Power BI**. It delivers real-time weather insights like temperature, humidity, air quality, and forecast for Indian cities including **Delhi**, **Bengaluru**, and **Ajmer**. The dashboard features a **custom UI designed in Figma** and integrates **live data via Weather API**‚Äîshowcasing the power of data storytelling and design.



## Purpose

- To demonstrate the use of **real-time data integration** in Power BI  
- To apply **data transformation and modeling** using Power Query and DAX  
- To highlight **storytelling with data** through visuals and layout  
- To create a **portfolio-ready project** suitable for interviews and business demos
